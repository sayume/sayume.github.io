Deploy node application with docker

Refer to the node official site, you can see how to hold a node app in an container, and we can also follow the steps provided by the official site of node.

First, you create your own node app, I created an express app named helloWorld as example. Then write the docker file for it, you can find a very short version of it on the official site:

The first line means you use node 1.0 to build the image, the second line means you expose port 3000 to the host machine.

Then go to the directory and use the following command to build the image "hello World":

    sudo docker build -t helloworld helloWorld/

And use this command to start the image,of course, you have to configure the start script and dependencies in the package.json file:

    docker run -it --rm --name helloWorld -p 3000:3000 helloworld

We start a container named helloWorld.
